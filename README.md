# Stock Trading Platform

A full-stack stock trading application built with modern web technologies, featuring real-time trading capabilities, portfolio management, and user authentication.

### Trading & Portfolio
- **Real-time Stock Trading**: Buy and sell stocks with live market data
- **Portfolio Management**: Track holdings, positions, and investment performance
- **Watchlist**: Monitor favorite stocks and market movements
- **Order Management**: Place and track buy/sell orders
- **Interactive Charts**: Visualize stock performance with Chart.js

### User Experience
- **User Authentication**: Secure signup/login with JWT and bcrypt
- **Responsive Design**: Mobile-friendly trading interface
- **Dashboard**: Personalized trading dashboard with portfolio overview
- **Landing Page**: Professional marketing pages (About, Products, Pricing, Support)

### Technical Features
- **Real-time Data**: Integration with live stock market APIs
- **CRUD Operations**: Full create, read, update, delete functionality
- **Data Visualization**: Interactive graphs for portfolio analytics
- **RESTful API**: Robust backend with Express.js and MongoDB
